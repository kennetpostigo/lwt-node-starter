# reason-node-starter

Quick project that sets up a basic project with reason-node using esy.

### Getting started

**Clone Repo**

```bash
git clone https://github.com/kennetpostigo/reason-node-starter.git
```

**Install**

```bash
esy install
```

**Build**

```bash
esy build
```

**Run Code**

```bash
esy make run
```
