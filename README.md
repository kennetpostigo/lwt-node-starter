# lwt-node-starter

Quick project that sets up a basic project with [lwt-node](https://github.com/kennetpostigo/lwt-node) using esy.

### Getting started

**Clone Repo**

```bash
git clone https://github.com/kennetpostigo/lwt-node-starter.git
```

**Install**

```bash
esy install
```

**Build**

```bash
esy build
```

**Run Code**

```bash
esy make run
```
